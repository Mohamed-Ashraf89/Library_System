# 📚 Library Management System (OOP)

A professional C# Console Application designed to demonstrate core **Object-Oriented Programming (OOP)** principles. This system manages various library assets like Books and Magazines with specific logic for borrowing and availability.

## 🚀 Technical Highlights
This project was built to practice and implement:
* **Abstraction:** Used abstract classes to define a blueprint for all library content.
* **Inheritance:** Created specialized classes (Book, Magazine) that inherit from a base content class.
* **Polymorphism:** Overrode methods to provide unique behavior for different media types.
* **Interfaces:** Implemented `IBorrowable` to separate logic for items that can be borrowed vs. those that are reference-only.
* **C# Features:** Used Lists, Constructors, and Type Casting (`is`/`as` operators).

## 🛠️ Built With
* **Language:** C#
* **Framework:** .NET Core / .NET Framework
* **IDE:** Visual Studio 2022

## 📂 Project Structure
* `Program.cs`: The entry point where the system logic is executed.
* `LibraryContent.cs`: Abstract base class for all items.
* `Book.cs` & `Magazine.cs`: Concrete implementations.
* `IBorrowable.cs`: Interface for borrowing functionality.
